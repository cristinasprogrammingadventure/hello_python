import matplotlib.pyplot as plt
from functools import reduce

# Define the list of numbers
num = [1, 2, 3, 4, 5, 6, 7, 8, 9]

# Guardar las variables por separado
results_1 = []
results_2 = []
results_3 = []

# Perform incremental reductions and store results
for i in range(1, len(num) + 1):
    partial_list = num[:i]
    result_1 = reduce(lambda x, y: x*2 + y*2, partial_list)
    result_2 = reduce(lambda x, y: x + y, partial_list, 4)
    result_3 = reduce(lambda x, y: x*2 + y, partial_list, 3)
    
    results_1.append(result_1)
    results_2.append(result_2)
    results_3.append(result_3)

# Generate time points
time_points = list(range(1, len(num) + 1))

#---------------------------------------------
# Vizualizar los resultados

plt.figure(figsize=(14, 8))

plt.plot(time_points, results_1, label='x*2 + y*2', linestyle='-', marker='o', color='blue')
plt.plot(time_points, results_2, label='x + y (start with 4)', linestyle='--', marker='s', color='green')
plt.plot(time_points, results_3, label='x*2 + y (start with 3)', linestyle='-.', marker='d', color='orange')

plt.xlabel('Time', fontsize=12)
plt.ylabel('Result', fontsize=12)
plt.title('Time Series of Reductions', fontsize=14)
plt.legend(loc='upper left', fontsize=10)
plt.grid(True, linestyle='--', alpha=0.5)
plt.xticks(fontsize=10)
plt.yticks(fontsize=10)

plt.tight_layout()
plt.show()

Las reducciones (o funciones de reducción) pueden ser herramientas en la programación para una variedad de aplicaciones en la vida real. Aquí hay algunos casos de uso comunes:¶

  • Procesamiento de Datos y Anlisis: Anlisis de Datos Financieros: En el anlisis financiero, las reducciones pueden utilizarse para calcular mtricas como el rendimiento de la inversin, el valor actual neto (VAN), o la tasa interna de retorno (TIR).

Anlisis de Series Temporales: Las reducciones pueden ser tiles para calcular estadsticas resumidas sobre series temporales, como la suma acumulativa de ventas diarias, el promedio mvil, o la tendencia general de los datos.

  • Ingeniera de Software y Optimizacin: Optimizacin de Algoritmos: Las reducciones pueden ayudar a simplificar algoritmos y mejorar la eficiencia computacional al reducir una lista de valores a un resultado nico.

Procesamiento de Datos en Paralelo: En entornos de procesamiento de datos distribuidos o en paralelo, las reducciones pueden ser tiles para reducir grandes conjuntos de datos en resultados manejables.

  • Modelado Matemtico y Cientfico: Clculos Matemticos y Cientficos: En el modelado matemtico y cientfico, las reducciones pueden ser utilizadas para calcular integrales, sumas de series, o realizar clculos estadsticos sobre datos experimentales.

  • Automatizacin de Tareas: Automatizacin de Procesos: Las reducciones pueden ser parte de scripts o programas que automatizan tareas repetitivas, como el procesamiento de archivos de datos, la generacin de informes, o la limpieza de datos.

  • Manipulacin de Datos en Ciencia de Datos y Machine Learning: Preprocesamiento de Datos: En ciencia de datos y machine learning, las reducciones pueden ser tiles para transformar y resumir conjuntos de datos antes de aplicar algoritmos de modelado.

Feature Engineering: Las reducciones pueden ser utilizadas para generar nuevas caractersticas (features) a partir de conjuntos de datos existentes, condensando la informacin relevante en un formato ms til para los algoritmos de aprendizaje automtico.
